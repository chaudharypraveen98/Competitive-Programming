## [22. min max](https://bigfrontend.dev/quiz/min-max)

The Three Golden Rules for this Quiz

1. **The Identity Elements**: If no arguments are given, Math.min() returns Infinity, and Math.max() returns -Infinity. This ensures that any subsequent number compared to these results will be correctly identified as smaller or larger. 
2. **Numeric Coercion**: All arguments passed to these functions are converted to numbers. 
3. **The Array Trap**: Neither Math.min() nor Math.max() accepts an array as a single argument. They expect individual numeric values.
