## [31. Math]()

### Approach
1. For Division, the specs says Division of a finite value by an infinity results in zero.
2. For Multiplication, the specs says Multiplication of an infinity by a zero results in NaN
3. Follow `BODMAS`

### Tips

X / 0 = Infinity

0 / 0 = NaN

-X / 0 = Infinity

Infinity * X = Infinity

Infinity * -x = -Infinity

Infinity + Infinity = Infinity

Infinity * Infinity = Infinity

Infinity - Infinity = NaN

Infinity / Infinity = NaN

0 * Infinity = NaN

0 / Infinity = 0
