## [14. Addition vs Unary Plus](https://bigfrontend.dev/quiz/Addition-vs-Unary-Plus)

### The Three Golden Rules for this Quiz
1. Unary Plus (+) Priority: A + placed directly before a value (e.g., + '2') is a unary operator. It forces that value into a number immediately before any other addition happens.
2. Binary Plus (+) Dual Nature: * If either side is a string, it performs string concatenation.If both sides are non-strings (numbers, booleans, null), it performs addition.
3. Primitive Numeric Values: 
   1. true --> 1 | false --> 0.
   2. null --> 0.undefined --> NaN.
